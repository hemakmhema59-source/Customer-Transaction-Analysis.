# Customer Transaction Analysis

## 📌 Project Overview
This project analyzes customer transaction data to identify patterns, detect anomalies, and generate insights for business decisions.

## 🎯 Objective
- Detect unusual transactions
- Ensure data integrity
- Provide useful business insights

## ⚙️ Steps Performed
1. Data Loading
2. Data Cleaning (removed null & duplicate values)
3. Data Analysis
4. Anomaly Detection
5. Visualization
6. Insights Generation

## 📊 Results
- Most transactions/customers are normal
- Few anomalies detected based on high values
- Helps in identifying suspicious behavior

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib

## ▶️ How to Run
```bash
pip install pandas matplotlib
python customer_transaction_analysis.py
